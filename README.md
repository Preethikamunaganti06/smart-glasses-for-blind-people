# Budget Friendly Smart Glasses for Blind People

## Overview
The Budget Friendly Smart Glass for Blind People project is designed to assist visually impaired individuals by providing real-time obstacle detection and feedback. The system uses ultrasonic sensors interfaced with an Arduino microcontroller to detect obstacles in the userâ€™s path and alerts them through audio signals. The solution focuses on affordability, simplicity, and portability while enhancing mobility and safety.

---

## Working Principle
Ultrasonic sensors continuously measure the distance to nearby obstacles. The Arduino processes this data and determines whether an object is within a predefined threshold distance. If an obstacle is detected, the buzzer provides an audio alert to warn the user in real time.
